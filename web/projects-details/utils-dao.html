Sobre

Biblioteca Java, para operações repetitivas de bancos de dados. O objetivo é abstrair consultas e operações de CRUD no banco de dados, a partir de uma API fluente, facilitando o desenvolvimento, e permitindo o enfoque nas regras de negócio.
